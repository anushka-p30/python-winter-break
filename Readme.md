Project 1: Resume parser and keyword extractor

## Problem
Manually scanning resumes to understand skills and content is time consuming. this project automates basic resume  analysis.

## Input
- currently a Resume text file (.txt)

## Output
- Total word count of the resume
- (Later) Important keywords from the resume

## How to Run 
- Place resume text in 'resume.txt'
- Run 'python parser.py'
